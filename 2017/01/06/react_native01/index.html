处理触摸事件

可点击控件，Touchable开头

  • TouchableHighlight来制作按钮或者链接
  • Android上还可以使用TouchableNativeFeedback,类似墨水涟漪的视觉效果
  • TouchableOpacity会在用户手指按下时降低按钮的透明度，而不会改变背景的颜色。
  • TouchableWithoutFeedback, 不显示任何视觉反馈。
